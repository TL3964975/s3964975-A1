A Software Engineer develops, modifies and implements new features to a software product which is either for a client of the business or for the organisation itself. They are also responsible for testing and identifying bugs or problems with the software which would then need to be fixed. They also work with the client and team members to come up with new ideas or features that could benefit the users of the software. These could range from features that could be necessary for the software to work, suggestions / feedback from clients or other departments in the organisation to assist in any improvements or features that may be necessary as well as quality-of-life changes which make using the software easier and more enjoyable for the user.

I find this type of job appealing as I would be able to either develop a new kind of software or implement new features which will be exciting for me especially knowing that other people will be implementing the software, I use in either their daily life or work. I am also used to talking to clients and pitching ideas like a sales pitch as it is like what I am currently doing at my current job. I also do prefer to work with a group and collaborating with others in my team to work to build or improve on a project for me is satisfying.

I would also need experience with coding and programming as I would need to ensure that there are no problems with the code so that the program or software works without any major bugs or problems. I would also need an understanding of algorithms.
