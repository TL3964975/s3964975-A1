They are also responsible for testing and identifying potential issues with the software which would then need to be fixed. They also work with the client and team members to come up with new ideas or features that could benefit the users of the software. These could range from features that may be necessary for the software to function such as suggestions and feedback from clients or other departments in the organisation as well as quality-of-life changes which make using the software easier and more enjoyable for the user.
